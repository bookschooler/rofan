### 2단계: CF 모델 자체의 치명적 문제 발견

조사 결과 네이버 웹툰 CF가 불가능한 이유 두 가지:

**문제 1: 별점 데이터가 구분 불가능**
- 네이버 웹툰 Kaggle OSMU 데이터셋 분석
- 전체 1,564개 작품 중 별점이 9.0~9.99점대에 91% 집중
- CF는 "이 유저가 좋아하는 것 vs 싫어하는 것"을 구분해야 작동
- 별점이 다 9점대면 모든 웹툰을 "다 좋아한다"고 해석 → 의미 없음

쏘피 질문: "별점이 다 9점대에 몰려있으면 네이버는 CF를 어떻게 적용한다는 거야?"
→ Claude 설명: 실제 네이버는 implicit feedback(클릭 수, 읽은 시간, 재방문 여부)을 사용. 이건 사용자 행동 로그라 비공개.

**문제 2: 실제 데이터 구조**
- 네이버 웹툰이 공개한 데이터: 작품 메타데이터(제목, 장르, 별점) → CF 불가
- 유저-작품 상호작용 데이터(누가 어떤 웹툰을 읽었나): 비공개
- 결론: Kaggle OSMU 데이터셋에도 유저 행동 데이터 없음

## 2. 분석 방향 결정 과정 (질문 → 답변 → 결정)

### 왜 신작 수를 지표로 쓰나

- 이상적인 지표: 조회수, 다운로드 수, 실매출 → 전부 비공개
- 실제 수집 가능한 지표: 작품 메타데이터(제목, 장르, 연재시작일)
- **근거**: 작가와 출판사는 인기 있는 장르에 새 작품을 냄. 로판 신작이 급증 = 시장 성장 신호.
- 이걸 시계열 데이터로 변환: "월별 로판 신작 수" 시계열

### 핵심 개념 설명 (쏘피가 물어봤던 것들)

**시계열 분석은 언제 쓰나?**
쏘피가 직접 질문함.
→ "데이터가 시간 순서로 나열되어 있고, 그 흐름에서 패턴(트렌드, 계절성)을 찾거나 미래를 예측할 때 사용"
→ 로판 신작 수를 월별로 나열하면 시계열 → 언제 급증했는지, 계절 패턴이 있는지 파악 가능

**STL 분해란?**
- Seasonal Trend decomposition using Loess
- 시계열을 3가지로 분리: 트렌드(T) + 계절성(S) + 잔차(R)
- 트렌드: 장기적 증가/감소 방향
- 계절성: 매년 반복되는 패턴 (예: 여름방학에 신작 많음)
- 잔차: 트렌드와 계절성으로 설명 안 되는 부분

**PELT 변동점 감지란?**
- Pruned Exact Linear Time (ruptures 라이브러리)
- 시계열에서 "구조적 변화가 일어난 정확한 날짜"를 찾는 알고리즘
- 예: 로판 신작 수가 2019년 10월에 갑자기 늘었다면 그 시점을 자동 감지

**ITS란?**
- Interrupted Time Series (중단된 시계열 분석)
- 특정 이벤트 전/후로 추세가 얼마나 바뀌었는지를 OLS 회귀로 검정
- 더미 변수로 이벤트 시점 인코딩 → p-value로 통계적 유의성 확인
- "재혼황후 웹툰 연재 후 로판 신작이 통계적으로 유의미하게 늘었나?"에 답할 수 있음

**Bass Diffusion 모델이란?**
- 신기술/문화 콘텐츠가 시장에 퍼지는 속도를 모델링
- 파라미터 p(광고 계수), q(입소문 계수), m(총 시장 크기)
- "로판 채택이 S자 곡선을 따른다면 포화점은 언제인가"에 답

**Prophet이란?**
- Facebook이 만든 시계열 예측 라이브러리
- 트렌드 + 계절성 + 특정 이벤트(공휴일 등) 효과를 자동 처리
- 95% 신뢰 구간 포함한 미래 예측치 출력
- 장점: 튜닝 없이도 어느 정도 잘 작동, 해석이 쉬움

## 9. 핵심 도메인 지식 요약

**로판(로맨스 판타지)이란?**
- 귀족/왕족 유럽풍 세계관 + 회귀(과거로 돌아감) / 빙의(다른 몸에 들어감) 설정
- 여성 주인공이 전생의 억울함을 복수하거나, 신분제 세계에서 성공하는 서사
- 20~40대 여성의 현실 대리만족 → "현실에서 못 하는 것"의 판타지

**IP 파이프라인이란?**
웹소설 원작 → 웹툰화 → 드라마/영화화 순서로 독자/시청자 규모 확대
- 각 단계마다 새 독자 유입 + 기존 팬 재소비
- 드라마 방영 → 웹소설 원작 판매 역증가 (선재 업고 튀어: 8.2배 증가)

**핵심 이벤트 — 재혼황후**
- 네이버 시리즈 웹소설 원작 → 2019년 10월 25일 네이버 웹툰 연재 시작
- 시즌4까지 진행, 글로벌 서비스
- 이 시점 이후 로판 신작 급증 가설 → Phase 2-3에서 통계로 검정 예정

**시장 규모 수치 (RESEARCH.md 출처 있음)**
- 2024년 웹소설 시장: 1조 3,500억원
- 로판 비중: 31.7% (2021년 기준, 1위)
- 카카오페이지 상위 50작품 중 72%가 로판
- 밀리의 서재: 2025년 6월 웹소설 탭 출시, 3년간 600억 투자
