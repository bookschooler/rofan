# Mann-Kendall 검정 — "오르는 방향이 통계적으로 맞나?" 확인하기

> **이 검정이 하는 일**: "로판 신작이 전반적으로 증가하는 추세가 맞나?"를 직선 가정 없이 검증하기
>
> **프로젝트에서 쓰는 곳**: Phase 2 (STL) — OLS 선형 회귀로 기울기 계산 후, 비선형 성장에서도 추세가 유효한지 추가 확인

**원논문(Tier 1)**: Mann, H.B. (1945). *Nonparametric tests against trend.* Econometrica, 13(3), 245–259.

---

## 쉬운 비유로 먼저 이해하기

계단이 있어요. 계단 높이가 일정하지 않아요 — 어떤 계단은 10cm, 어떤 계단은 30cm, 어떤 계단은 5cm예요.

OLS 선형 회귀는 "이 계단들의 평균 높이가 양수냐?"를 물어봐요. 직선으로 가정하고요.

Mann-Kendall은 이렇게 물어봐요:

> "계단이 높이가 들쑥날쑥해도, **전체적으로 올라가는 방향이 맞나**?"

계단 높이(기울기 크기)는 몰라도 돼요. 방향이 일관되게 위로 향하는지만 봐요.

로판으로 바꾸면:
- 어떤 달은 신작이 확 늘고, 어떤 달은 줄고, 어떤 달은 조금 늘어도
- **전체적으로 2013년보다 2025년에 신작이 더 많아지는 방향이 일관되게 맞나?**

---

## OLS vs Mann-Kendall — 언제 뭘 쓰나?

| 상황 | 추천 |
|------|------|
| 데이터가 직선처럼 일정하게 증가 | OLS 선형 회귀 |
| 처음엔 느리다가 나중에 급증 (S자 곡선) | Mann-Kendall |
| 이상치(갑자기 튀는 달)가 많을 때 | Mann-Kendall |
| "얼마나 빠르게 성장하나" 수치가 필요할 때 | OLS |
| "증가/감소 방향이 확실한가"만 알면 될 때 | Mann-Kendall |

> **왜 로판에서 둘 다 쓰나?** 로판 시장은 S자 성장 패턴이 의심되는데, OLS만 쓰면 초기 완만한 구간에서 추세가 과소평가될 수 있어요. Mann-Kendall로 방향 자체를 따로 확인해요.

---

## 작동 원리 (수식 없이)

데이터 포인트가 n개 있을 때, 가능한 모든 시간 쌍 (i, j) — i가 j보다 앞에 오는 — 을 비교해요.

- 나중 값이 더 크면: **+1점** (증가)
- 나중 값이 더 작으면: **-1점** (감소)
- 같으면: **0점**

이 점수를 전부 더한 게 **S 통계량**이에요.

```
S가 크게 양수 → 대부분의 쌍에서 나중 값이 더 큼 → 증가 추세
S가 크게 음수 → 대부분의 쌍에서 나중 값이 더 작음 → 감소 추세
S가 0에 가까움 → 패턴 없음
```

그리고 S가 우연히 이 크기가 나올 확률(p-value)을 계산해요.

---

## 수식 (딱 핵심만)

$$\tau = \frac{S}{\text{가능한 쌍 수}}$$

이걸 **Kendall's Tau**라고 해요. -1에서 +1 사이 값이 나와요.

- **τ = +0.8**: 80%의 쌍에서 "나중이 더 크다" → 강한 증가 추세
- **τ = -0.3**: 30%의 쌍에서 "나중이 더 작다" (일관되지 않음) → 약한 감소 추세
- **τ = 0**: 방향 없음

---

## 코드로 보기

```python
import pymannkendall as mk

result = mk.original_test(monthly_counts)  # 월별 신작 수 배열

print(result.trend)   # 'increasing' / 'decreasing' / 'no trend'
print(result.p)       # p-value (0.05보다 작으면 추세가 통계적으로 유의미)
print(result.Tau)     # Kendall's Tau (-1~+1, 방향과 강도)
```

**결과 읽는 법**:
```
trend=increasing, p=0.000001, Tau=0.72
→ "증가 추세가 맞고, 우연일 확률 0.0001%, 72%의 쌍에서 나중이 더 큼"
```

## 핵심 가정 — "이게 맞아야 Mann-Kendall을 쓸 수 있어요"

| 가정 | 쉬운 말 | 깨지면? |
|------|--------|--------|
| 데이터가 시간 순서로 독립적이다 | 이번 달 신작 수가 지난달에 영향 안 받음 | 강한 자기상관이 있으면 p-value 과소 추정 (modified MK 사용) |
| 최소 8~10개 이상의 관측값 | 분석 기간이 충분히 길어야 함 | 데이터가 너무 적으면 검정력이 낮아짐 |

> **자기상관이 있으면?** 월별 시계열은 보통 자기상관이 있어요 (이번 달 많으면 다음 달도 많을 가능성). 이 경우 `mk.hamed_rao_modification_test()` 사용.

## 이럴 때 Mann-Kendall이 실패해요

### ❌ 실패 1: 계절성이 제거되지 않은 데이터에 바로 적용
```python
# 틀린 사용: 원본 시계열에 바로 적용
result = mk.original_test(raw_monthly_data)
```
→ 매년 1월에 높은 계절성이 "증가 추세"로 잘못 잡힘.
→ **반드시 STL로 계절성 제거 후 추세 성분(trend)에 적용하거나, seasonal MK 사용**

### ❌ 실패 2: 단조(monotonic) 추세가 없는 U자형 데이터
- "처음엔 늘다가 줄어드는" 형태는 Mann-Kendall이 "추세 없음"으로 판정할 수 있어요
- 이 경우 구간을 나눠서 각각 검정해야 함

## 로판 프로젝트에서 구체적으로 어떻게 쓰나요?

**분석 흐름**:
1. STL로 트렌드 성분 추출
2. OLS로 기울기 계산 (연간 증가폭 수치)
3. Mann-Kendall로 "이 증가 방향이 우연이 아니다"를 비선형 가정 없이 추가 검증
4. 두 검정 모두 p < 0.05 → 결론 강화 ("선형 가정을 했을 때도, 안 했을 때도 증가 추세 확인")

---

## 면접에서 "왜 Mann-Kendall을 추가로 썼나요?" 질문이 나오면

> "OLS 선형 회귀는 데이터가 직선처럼 증가한다는 가정이 있어요. 로판 시장은 초기 완만한 성장 후 급증하는 S자 곡선 패턴이 의심됐는데, 이럴 때 OLS만 쓰면 추세를 과소평가할 수 있어요. Mann-Kendall은 성장 형태에 대한 가정 없이 방향의 일관성만 검정하는 비모수 방법이라 추가 검증으로 사용했고, 두 방법 모두 p<0.001로 증가 추세를 확인해 결론의 강건성을 높였습니다."
